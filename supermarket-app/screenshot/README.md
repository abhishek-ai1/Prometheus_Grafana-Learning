# Screenshots

This directory holds PNG images of the application UI and monitoring dashboards used throughout the documentation. The current files are actual captures of the running system; feel free to update any of them by overwriting with a new PNG file, retaining the same filename so the root README continues to display it.

To update images:

1. Save a new screenshot with the same name as the file you wish to replace (remaining in this folder).
2. Commit the change and the project README will automatically show the updated image.
