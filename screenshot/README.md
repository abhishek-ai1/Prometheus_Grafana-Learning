# Screenshots

This directory is intended to hold PNG images of the application UI and monitoring dashboards used throughout the documentation.

Currently the files are placeholders containing only a text marker (`[screenshot image placeholder]`).

To provide real screenshots:

1. Replace the contents of each `.png` file with the actual binary image (or delete and add new ones with the same name).
2. The README at the repository root will automatically reference these names; updating the files will make them appear in the documentation.
